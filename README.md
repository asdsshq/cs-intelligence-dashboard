# CS Intelligence Dashboard

CS Intelligence Dashboard is an analytics project for Counter-Strike match analysis, team comparison, player performance tracking, and pre-match insights.

## Features

- Team comparison
- Player statistics
- Recent form analysis
- Map pool analysis
- Head-to-head comparison
- Pre-match analytical summary

## Tech Stack

- Python
- pandas
- Streamlit
- Plotly
- SQLite / PostgreSQL

## Project Status

In development.
